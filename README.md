# RetroRPG

RetroRPG is a pixel-art role-playing game currently under active development.  
The project focuses on building a clean and extensible game architecture, with the long-term goal of running the game on custom-designed hardware.

---

## Overview

This repository contains the core game implementation, including gameplay systems, assets, and supporting infrastructure.  
Development is organized using a structured workflow with dedicated branches, issues, and project tracking.

---

## Vision

Beyond software development, this project aims to explore full-stack game creation — from gameplay to hardware.

Planned objectives include:

- Designing a custom PCB using Altium Designer  
- Selecting and integrating a microcontroller  
- Developing a minimal custom operating system  
- Running the game directly on dedicated hardware  
- Designing and 3D printing a custom enclosure  

The goal is to create a compact, self-contained handheld system inspired by classic devices such as the Game Boy.

## Development Workflow

The project follows a structured branching model:

- `main` — stable, production-ready code  
- `dev` — active development branch  
- `feature/*` — isolated feature development  

All changes are introduced via pull requests and reviewed before merging.
